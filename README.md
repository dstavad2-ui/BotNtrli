# Naturlig AI Telegram Bot

En Telegram-bot til harm reduction og sikkerhed for unge.

## Installation

1. Klon dette repo.
2. Kør `bash scripts/setup.sh` for at installere afhængigheder.
3. Kør `bash scripts/deploy.sh` for at starte serveren og indstille webhook.

## Brug

- `/start`: Velkomstbesked.
- `/help`: Hjælp.
- `/guide <substance>`: Få harm reduction guides.
